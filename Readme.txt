This is a thermal label printing tool, to help end users print a set of pre-defined formats of labels, with several changing fields such as datetime, serial number and so on.

This tool internally relies on Bartender SDK(and thus Bartender automation installation).

The tool will look for Bartender label files(*.btw) in pre-defined folders, and display them
in the UI. Each folder is a class of rule which specifies the formats of additional changing fields
values. The user can trigger print either by the button or by signaling the serialport.
